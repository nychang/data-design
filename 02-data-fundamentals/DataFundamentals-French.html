Données : Principes de base

Quand vous faites la cuisine, vous suivez généralement une recette. C'est une succession d'étapes réalisées dans un ordre donné. Vous commencez par réunir vos ingrédients. Vous les manipulez ensuite selon ce que la recette exige. Enfin, vous montez vos assiettes avec soin, car tout bon chef vous dira combien ce qui compte, en gastronomie, c'est la présentation.

La visualisation de données ressemble beaucoup à la cuisine. Vous devez d'abord collecter vos données. Vous devez ensuite les traiter. Vous devez enfin les visualiser.

En cuisine, il est utile de bien connaître ses ingrédients afin de savoir ce qui est possible de faire avec chacun : le sel et le sucre ont le même aspect, mais ne jouent pas le même rôle!

Dans cette première section, nous allons considérer les données comme les ingrédients de base d'une bonne visualisation. Nous allons vous familiariser avec les différents types de données afin que vous soyez plus à l'aise avec ceux-ci dans les étapes ultérieures du livre.
